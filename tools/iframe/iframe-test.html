🔧 Technické detaily

Provedené změny:

  • Přidány HTTP hlavičky pro povolení iframe embedování
  • Vytvořen Express server s middleware pro správné hlavičky
  • Odstraněny X-Frame-Options omezení
  • Nastavena permisivní Content-Security-Policy
  • Přidány CORS hlavičky pro cross-origin požadavky

Pro nasazení na Railway:

  1. Commitněte všechny změny do Git
  2. Pushněte na Railway
  3. Railway automaticky použije npm start příkaz
  4. Aplikace bude dostupná s podporou iframe
